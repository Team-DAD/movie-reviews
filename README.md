## Development

### Installing Dependencies

From within the root directory:

```sh
npm install -g webpack
npm install
```

## Seeding database

From within root:
npm run seed-db

If error try from mysql shell:
SHOW GLOBAL VARIABLES LIKE 'local_infile';

If false, set with
SET GLOBAL local_infile = true;
